# Data Analyst Portfolio

Welcome to my Data Analyst Portfolio! This repository showcases my projects and skills in data analysis, machine learning, predictive modeling, and business intelligence. The portfolio demonstrates hands-on experience with SQL, Python, regression analysis, cohort analysis, A/B testing, clustering, classification models, and AI agent development.

Check out my portfolio here: [Data Analyst Portfolio](https://fiqcodes.github.io/DataAnalystPortfolio/)

## Portfolio Overview

This portfolio includes the following projects:

1. **ABC Company Analysis** – Descriptive statistics, EDA, and linear regression to predict property prices and provide business recommendations for revenue growth.

2. **Campaign's Effect on Sales** – EDA, A/B testing using t-tests, and linear regression to predict sales impact from marketing campaigns.

3. **TheLook e-Commerce Analysis** – SQL-based cohort analysis and BCG Matrix evaluation to analyze user retention and product category growth.

4. **User Segmentation & Churn Prediction** – K-Means clustering, logistic regression, EDA, and benefit–cost analysis for targeted campaigns.

5. **Predict Customer Conversion** – XGBoost classification, cohort analysis, and EDA to predict customer conversion and improve retention.

6. **SmartLook AI Agent** – AI-powered data analyst and SQL agent built with LangGraph, few-shot learning, and structured reasoning flows, featuring a React interface for automated insight generation.

## Skills Demonstrated

* **Statistics & Machine Learning**: Descriptive statistics, regression analysis, A/B testing, clustering, classification, predictive modeling
* **Data Analysis & Visualization**: Exploratory data analysis (EDA), cohort analysis, business insight generation
* **SQL & Databases**: Complex queries, window functions, cohort analysis
* **Python & AI Development**: K-Means, logistic regression, XGBoost, AI agent development
* **Business Intelligence**: Revenue growth analysis, campaign ROI analysis, product strategy recommendations
